# ProyectoGitDAM
Para la documentación de este proyecto primero voy a explicar cómo funciona el programa de prueba creado y, después, voy a seguir los puntos señalados en la actividad.

## Programa y explicación
Programa sencillo en Java que muestra un mensaje para pedir un texto que guardará y devolverá 2 veces seguidas.

### Ejecución
Ejecutar la clase PRUEBA1_1 desde Eclipse.

## Descripción del funcionamiento
### Creación de cuenta en GitHub: 
Ya tenía cuenta creada anteriormente, pero bastaría con introducir correo, contraseña y validar después de crear la cuenta.
### Creación de repositorio remoto:
Desde mi cuenta creo el repositorio llamado ProyectoGitDAM, viendo que esté en visibilidad pública y no añada un README, ya que este será el README que adjuntaré posteriormnente.
### Conexión del proyecto local con GitHub:
1. Buscar la URL del repositorio y copiarla.
2. Hacer clic derecho sobre el proyecto.
3. Seleccionar **Team** -> **Push Branch 'main'**.
4. Configurar el repositorio metiendo la URL y configurando usuario y token.
5. Enviar los commits.
